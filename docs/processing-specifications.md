# 中药饮片炮制规范

> 依据《中华人民共和国药典》（2020年版）四部通则 0213 及各部药材炮制项整理

---

## 一、炮制总则

中药饮片炮制系指将中药材通过净制、切制或炮炙等操作，制成一定规格的饮片，以适应医疗要求及调配、制剂的需要，保证用药安全和有效。

### 1. 炮制目的

| 目的 | 举例 |
|------|------|
| 降低或消除毒性/副作用 | 川乌煮制减毒、半夏姜制减毒 |
| 改变或缓和药性 | 地黄生寒熟温、何首乌生泻熟补 |
| 增强疗效 | 蜜炙款冬花增强润肺止咳、醋延胡索增强止痛 |
| 改变作用趋向 | 酒制升提、盐制入肾、醋制入肝 |
| 便于调剂和制剂 | 切制成一定规格、粉碎 |
| 矫味矫臭，便于服用 | 麸炒、蜜炙、酒炙等 |
| 纯净药材，保证质量 | 除去杂质、非药用部位 |

### 2. 炮制用水

- 饮用水：净制、润药、蒸煮等均使用饮用水
- 制药用水应符合《中国药典》相关标准

### 3. 辅料

**液体辅料：**

| 辅料 | 规格要求 | 常用量 |
|------|----------|--------|
| 黄酒 | 含醇量 15%~20% | 每100kg药材用10~20kg |
| 米醋 | 含醋酸 ≥3.5% | 每100kg药材用15~30kg |
| 炼蜜 | 含水量 ≤18% | 每100kg药材用20~30kg |
| 生姜汁 | 鲜姜榨汁 | 每100kg药材用12.5~25kg |
| 盐水 | 食盐（化工商标） | 每100kg药材用2~3kg |

**固体辅料：**

| 辅料 | 规格 | 作用 |
|------|------|------|
| 麦麸 | 小麦加工副产品 | 健脾和中、矫味 |
| 灶心土 | 久经柴草熏烧 | 温中止泻 |
| 滑石粉 | 白色细粉 | 使药物质地酥脆 |
| 蛤粉 | 文蛤贝壳煅制粉碎 | 清热利湿 |
| 河砂 | 洁净河砂 | 使药物质地酥脆 |

## 二、净制通则

中药材必须经过净制，方可进行切制或炮炙。

### 1. 净制方法

- **挑选**：除去杂质及非药用部位
- **筛选**：筛去砂石、碎屑
- **风选**：利用风力除去杂质
- **水选**：洗净泥土、盐分
- **剪、切、削**：除去残留的非药用部位

### 2. 去除非药用部位要求

| 部位 | 要求 |
|------|------|
| 芦头（根茎类） | 人参、桔梗等去芦头 |
| 皮（果实类） | 桂圆去壳、桃仁去皮 |
| 心（根皮类） | 远志去木心 |
| 核（果实类） | 山茱萸去核 |
| 毛（叶类） | 枇杷叶去毛 |
| 头尾足翅（动物类） | 斑蝥去头足翅 |

---

## 三、切制通则

### 1. 切制前软化

**常用软化方法：**

- **淋法**：气味芳香、含挥发油药材（薄荷、荆芥等），喷淋清水后稍润
- **洗法**：质地松软的药材（紫菀、陈皮等），快速洗涤
- **泡法**：质地坚硬的药材（大黄、何首乌等），清水浸泡至内外湿度一致
- **润法**：大多数药材适用，使水分徐徐渗入组织内部
- **蒸法**：含粘液质、遇水易发粘的药材（黄精、天麻等），蒸软后切制

### 2. 饮片类型与规格

| 类型 | 厚度 | 适用药材 |
|------|------|----------|
| 薄片 | 1~2mm | 根茎类质地坚实者（当归、白芍、川芎） |
| 厚片 | 2~4mm | 根茎类质地疏松者（黄芪、党参、白术） |
| 斜片 | 2~4mm | 长条形根茎（甘草、桂枝） |
| 直片 | 2~4mm | 形状不规则者（大黄、何首乌） |
| 丝 | 细丝2~3mm，宽丝5~10mm | 皮类、叶类（陈皮、黄柏） |
| 段 | 10~15mm | 全草类、茎枝类（麻黄、益母草） |
| 块 | 1~2cm³ | 质地坚实者（葛根、茯苓） |
| 粉末 | 药典规定细度 | 不宜切制的药材 |

### 3. 干燥

- 切制品应使用适宜方法干燥，温度一般不超过 **60℃**
- 含挥发油的药材干燥温度不超过 **50℃**
- 含水量应符合《中国药典》各药材项下规定

## 四、炮炙通则

### （一）炒法

#### 1. 清炒（不加辅料）

| 类型 | 火力 | 程度 | 举例 |
|------|------|------|------|
| 炒黄 | 文火/中火 | 表面微黄/有香气 | 炒山楂、炒麦芽 |
| 炒焦 | 中火/武火 | 表面焦褐/内部深黄 | 焦山楂、焦神曲 |
| 炒炭 | 武火 | 表面焦黑/内部焦褐（存性） | 地榆炭、侧柏炭 |

> 炒炭须"炒炭存性"——外部炭化、内部保有原药气味

#### 2. 加辅料炒

**麸炒：** 麸皮用量一般为 **每100kg药材用麸皮10~15kg**
- 增强健脾和胃作用
- 缓和药性、矫味矫臭
- 例：麸炒白术、麸炒山药、麸炒枳壳

**土炒：** 灶心土用量一般为 **每100kg药材用灶心土25~30kg**
- 增强温中止泻作用
- 例：土炒白术、土炒山药

**米炒：** 大米用量一般为 **每100kg药材用大米20kg**
- 增强补中益气
- 例：米炒党参

**砂炒：** 河砂用量不拘，炒至药材酥脆
- 使药物质地酥脆，易于煎出
- 降低毒性（马钱子砂炒减毒）

### （二）炙法

将药材与液体辅料拌炒的炮制方法。

| 炙法 | 辅料 | 关键操作 | 举例 |
|------|------|----------|------|
| 酒炙 | 黄酒 | 闷润至吸尽，文火炒干 | 酒当归、酒川芎、酒黄芩 |
| 醋炙 | 米醋 | 闷润至吸尽，文火炒干 | 醋延胡索、醋香附 |
| 盐炙 | 食盐水 | 闷润至吸尽，文火炒干 | 盐黄柏、盐知母 |
| 姜炙 | 姜汁 | 闷润至吸尽，文火炒干 | 姜黄连、姜竹茹 |
| 蜜炙 | 炼蜜 | 蜜稀释后拌匀，炒至不粘手 | 炙黄芪、炙甘草、炙麻黄 |
| 油炙 | 食用油 | 炸酥或炒制 | 油酥蛤蚧 |

**炙法操作要点：**
1. 液体辅料应充分与药材拌匀，闷润至辅料被吸尽
2. 炒制火力不宜过大，防止焦化
3. 蜜炙品应摊凉后密封保存，防止吸潮

### （三）煅法

将药材直接或间接煅烧的方法。

| 类型 | 操作 | 举例 |
|------|------|------|
| 明煅 | 直接煅烧至红透 | 煅石膏、煅牡蛎 |
| 煅淬 | 煅至红透后入液体辅料淬 | 煅自然铜（醋淬） |
| 扣锅煅 | 密闭高温煅烧 | 血余炭、棕榈炭 |

### （四）蒸煮法

| 方法 | 操作 | 举例 |
|------|------|------|
| 蒸 | 置蒸锅/蒸笼内蒸透 | 熟地黄、制黄精 |
| 煮 | 置水中煮至特定程度 | 川乌煮至无白心 |
| 燀 | 沸水短时间烫过 | 燀苦杏仁（去皮） |

### （五）复制法

将药材加入辅料反复炮制的方法，主要用于毒性药材。
- 半夏：清水浸泡→矾水浸泡→姜矾共煮
- 天南星：同复制法

### （六）发酵法

在适宜温湿度下利用微生物发酵。
- 六神曲：面粉、苦杏仁、赤小豆、鲜青蒿等发酵制成

### （七）发芽法

将成熟种子置适宜条件下萌发幼芽。
- 麦芽：大麦发芽至芽长约0.5cm
- 谷芽：粟谷发芽至芽长约0.5cm

## 五、炮制用水与火候通则

### 火力

- **文火**：小火，温度约 100~120℃
- **中火**：中等火力，温度约 120~180℃
- **武火**：大火，温度约 180~240℃

### 各类炮制常用火候

| 炮制方法 | 常用火力 | 时间 |
|----------|----------|------|
| 炒黄 | 文火~中火 | 至香气逸出 |
| 炒焦 | 中火 | 至表面焦褐色 |
| 炒炭 | 武火 | 至表面焦黑色（存性） |
| 炙法（酒/醋/盐/姜） | 文火 | 炒干至规定程度 |
| 蜜炙 | 文火 | 炒至不粘手 |
| 麸炒 | 中火 | 至表面深黄色 |
| 煅制 | 武火 | 至药材红透 |

## 六、贮藏通则

| 类型 | 要求 |
|------|------|
| 一般饮片 | 干燥、通风、防潮 |
| 含挥发油饮片 | 密闭、阴凉（如薄荷、当归） |
| 含糖类饮片 | 防潮、防虫（如枸杞子、党参） |
| 蜜炙饮片 | 密闭、阴凉（易吸潮发粘） |
| 盐炙饮片 | 密闭、防潮（易吸潮变软） |
| 煅制品 | 密闭、防潮 |
| 有毒饮片 | 专柜专锁、专人管理 |
| 鲜药 | 保鲜、每日翻动 |

## 七、质量管理

### 检查项目

| 项目 | 指标 |
|------|------|
| 水分 | 按药典各品种项下规定 |
| 灰分 | 总灰分、酸不溶性灰分 |
| 浸出物 | 按药典各品种项下规定 |
| 含量测定 | 有效成分含量 |
| 微生物限度 | 符合药典规定 |

### 注意事项

1. 炮制过程中应严格控制 **温度**、**时间**、**辅料用量**
2. **"炒炭存性"** — 不得全部炭化，须保留原药固有气味
3. **"逢子必炒"** — 种子类药材多需炒制，利于有效成分煎出
4. **"知母、黄柏多盐炙"** — 盐炙入肾增强滋阴降火
5. **"当归、川芎多酒炙"** — 酒炙增强活血通经
6. **"大黄生熟不同"** — 生大黄泻下力强，制大黄缓泻，大黄炭止血
7. **"地黄九蒸九晒"** — 熟地黄制法传统有"九蒸九晒"之说，药典为单蒸或加黄酒蒸至内外漆黑
